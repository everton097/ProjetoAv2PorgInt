Este repositório serve para a avaliação 02 de programação para internet I.

```
Continuando o trabalho iniciado na etapa 1 da avaliação:

Selecione uma paleta de cores do site https://colorhunt.co/
Utilizando a paleta de cores selecionada, crie uma folha de estilos (CSS) padrão para seu site, todos os elementos de todas as suas páginas devem ser estilizados;
Crie uma segunda folha de estilos para o seu site em que os mesmos elementos da primeira são estilizados, mas agora em um modo "escuro";
Adicione um botão em cada página do seu site para que seja possível alterar o CSS entre o modo padrão e o modo escuro;
Grave um vídeo de no máximo 10 minutos mostrando a página criada e mostrando no código como cada um dos critérios do enunciado foi atendido;
Envie o vídeo para o Youtube (caso queira mais privacidade, configure o mesmo para estar "não listado");
Envie aqui no Moodle o link do vídeo para o Youtube;
Envio aqui no Moodle o link para o local onde os arquivos do site estão armazenados (Google Drive, Github, W3Spaces, etc.).

Avaliação:

Seu trabalho deve ser original, cópias (plágios) receberão nota zero;
Trabalhos não enviados (e.g., salvo como rascunho no Moodle), receberão nota zero;
Trabalhos não enviados no Youtube receberão nota zero;
Vídeos com mais de 10 minutos receberão nota zero;
Vídeos privados, links quebrados e situações similares que impeçam que o vídeo seja assistido receberão nota zero;
O site estilizado com a paleta de cores padrão vale 50% da nota;
O botão e o funcionamento que permite trocar para a paleta de cores escura vale 50% da nota.
